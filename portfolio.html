My Projects

Hoop City Manager

An in-depth basketball simulation card game that I built in Unity. Designed custom graphics and 3D models. Created a back story for the game. Used firebase for cloud services. Developed AI to control opposing team managers. Released on Steam for PC and Google Play Store for Android.

Tejas Shop

A marketplace platform built for me and friends to sell items. Developed with Django framework. Deployed using Azure Web App plus Postgres database. Used azure cloud storage services. Also used Stripe API, PayPal API and Sendgrid API. Designed to be as cost efficient as possible.

Rivals Soccer

A soccer adventure game that I developed in Unity (WIP). Players search for and recruit AI players to compete against increasingly difficult opponents. Developed an in-depth traditional AI system for controlling opposing players. Designed and collected stylized assets for use in game. Added controller support. Used UI Toolkit.

Golfers Vs Zombies

Another game I developed in Unity. Used Machine Learning algorithms from Accord.Net to develop a dangerously accurate Zombie Golfer AI. Created custom models and other assets. Optimized performance for mobile and tested extensively on Android.

Hoop Today

A twitter clone for pick-up basketball. Developed during March Madness and decided to include a game predictor. Correctly predicted several unpredictable upsets. Added Machine Learning models developed with scikit-learn to use for fun prediction quizzes about the NBA (which player are you quiz). Deployed with Azure.

Islandia

Vintagetejas/Texscloset

3D Pop Shop

NBA 2K Rosters

Developed a program to generate NBA Rosters similar to NBA 2K. Collected, cleaned, and manipulated data to create ratings for each player in different categories. Utilized the rosters to generate probabilites of on-court events for simulation in Hoop City Manager.
